<p>ChatApp is a project that improve my Android Development knowledge. This app comprises use of Stack Navigation, Firebase Authentication/Database/Storage/Messaging/ErrorReporting, Moment component and Modal.</p>
<ul>
<li>App starts with Login Page. If user has a recognzied account, it is able to type usermail and password for login.</li>
<li>If the user insert invalid e-mail or password, it alerts error type according to made error.(It is checking with Firebase/auth during login process.)</li>
<li>If the user doesnt have an recognized account, the user can click "Sign Up" button in Login Page and redirect Sign Up page. The user must be filled input places. After click button of Create Account, account will be created and redirect to login page.</li>
<li>After successfull login, the user must choose a channel where she/he want send message.</li>
<li>It is available switch btwn channels with click related button on TopBar.</li>
<li>The user can display all messages with time sequence.</li>
<li>On the top right of the page, there is a log-out button.</li>
</ul>

<p>To use this application, the packages for Navigation, Moment, Modal, Vector Icons and Firebase should be installed/identified with prior to running the app. From your command line:</p>
<pre><code>npm install @react-navigation/native
For expo :
expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked-view
For bare React-native:
npm install react-native-reanimated react-native-gesture-handler react-native-screens react-native-safe-area-context @react-native-community/masked-view

npx pod-install ios(for IOS)

npm install @react-navigation/stack

npm install react-native-modal

npm install react-native-vector-icons

npm install moment
</code></pre>
<p>Note: Firebase is required for simulate real-time database. Before build up your project, need visit firebase page and identify your project with firebase project.</p>
